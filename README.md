# multi-tenant-iac

Deploy a REST API to a Virutal Server (VM) using different cloud providers with Infrastrucutre as Code and Ansible for provisioning. 

Multi tenant deployment using Terraform and Ansible.

- (AWS) Amazon Web Services 
- (Azure) Microsoft
- (GCP) Google Cloud

### Setup Python and Ansible stuffs
- `pip install -r requirements.txt`
- `ansible-galaxy install -r requirements.yml`
